Aula 008 - GITHUB e CSS.

  • Apresentacao do GITHUB:

    - Cadastro.
    - Funcoes.
    - Utilidades.
    - Licencas.

  • Introducao ao CSS:

    - Comandos de estilizacao.
    - Escreva dentro da secao "style".
    - Especifique o elemento (Ex: "H1") a ser alterada.
    - Utilize chaves "{}" para abrir e fechar os comandos.
    - Use a TAG "background" para definir o fundo do site.
    - Use a TAG "Border" para criar borda ao redor de algum elemento.
