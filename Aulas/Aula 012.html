  • Utilizacao de Metas, TAGs e Atributos para o site:

    - TAG "Span": Cria uma secao de conteudo um ao lado do outro. Frequentemente usada para dividir textos.
    - TAG "nav": Utilizada para facilitar a navegacao dentro de uma pagina.
    - TAG "main": Utilizado para definir o conteudo principal de uma pagina.
    - TAG "footer": Utilizado para definir o rodape.
    - TAG "div" cria uma secao uma abaixo da outra. Frequentemente usado para separar os conteudos em uma pagina.
    - TAG de arquivo CSS "*": Utilizada para restaurar toda a estilizacao de uma pagina em diferentes navegadores.
    - Atributo "list-style": Usado para definir o estilo de uma lista.
    - Atributo "Flex-wrap": Ajusta o conteudo quebrando-o para o tamanho da janela.
    - Atributo "Display: Block": Impede que um elemento fique na mesma linha.
    - Atributo "Display: Inline": Permite que um elemento seja exibido na mesma linha.
    - Atributo "Display: inline-block": Permite que um elemento fica um ao lado do outro com informacoes de altura e largura.
    - Atributo "Dislay: flex": Permite organizar os elementos da forma desejada.
    - Atributo "text-decoration": Usada para estilizar a a visualizacao do hiperlink.
    - Atributo "a:hover": Usado para estilizar um hiperlink ao passar o ponteiro do mouse.
    - Atributo "Text-align": Usado para alinhar um texto na pagina.
    - Atributo "Padding": Preenchimento de um conteudo.
    - Atributo "Margin": Espacamento da margem.
    - Atributo "flex-direction: row": Usado para definir um elemento um ao lado do outro.
    - Meta "viewport": Usado para adicionar novas unidades de medida. Frequentemente usada para modificar a exibicao de um conteudo na janela do usuario.
    - Atributo "ID": Especifica um nome especifico a um DIV para definir caracteristicas e chamados unicos a ele.
    - Para identificar uma classe em CSS, escreva "." e o nome dela.
    - Para identificar um ID em CSS, escreva "#" e o nome dele.
