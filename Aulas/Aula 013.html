Aula 013 - Projeto PETSHOP

  • Utilizacao de Metas, TAGs e Atributos para o site:

    - TAG-Atributo "Position: fixed": Deixa algo fixo na sua pagina.
    - TAG-Atributo "Z-index: numero": Define a prioridade de um elemento sobrepor outro.
    - TAG-Atributo "Tab-index: numero": Define a ordem de selecao de um elemento ao usar movimentacao por TAB.
    - TAG-Atributo "Posicao desejada: numero": Define a posicao absoluta de um elemento na pagina.
    - TAG-Atributo "Border-radius: x x x x": Define a curvatura dos cantos de algum elemento.
    - TAG-Atributo "RGBA( x x x x)": Define as cores de um elemento, e a sua opacidade.
    - TAG-Atributo "Cursor: tipo desejado": Define o tipo do cursor do mouse.
    - TAG-Atributo CSS "Box-sizing: border-box": Define um tamanho absoluto para um elemento.
    - TAG-Atributo CSS "'Nome elemento': after/before": Permite adicionar conteudos antes e depois de uma parte especifica.
    - TAG-Atributo "Table td:first-child": Define uma propriedade especifica para o primeiro elemento da coluna.
    - TAG-Atributo "Table td:last-child": Define uma propriedade especifica para o ultimo elemento da coluna.
    - TAG-Atributo "Table td:first-child": Define uma propriedade especifica para o primeiro elemento da coluna.
    - TAG-Atributo "Table td: nth(x)": Define uma propriedade especifica para o elemento especificado da coluna.
    - TAG-Atributo "Table tr:first-child": Define uma propriedade especifica para o primeiro elemento da linha.
    - TAG-Atributo "Table tr:first-child(odd/even)": Define uma propriedade especifica para os elementos impares/pares da linha.
